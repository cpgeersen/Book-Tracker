# 2.10 - Frontend Scripting for Local Search

### Main Objective:
Implement the scripts for receiving JSON results from the backend to populate the frontend local search page.

Note: Remember that we will be using infinite scrolling for pagination.

For this task do not worry about sending or reading information from a route. Act like the JSON given is a received
result. You can use your own constructed JSON if it follows this form. For now do not send a GET request and read book
information in the JSON schema given to populate a page. Once backend and frontend integration begins we will worry about
using routes, which will be easy since the JSON results will be in the same schema and will just need to read the result.

### JSON Schema:
