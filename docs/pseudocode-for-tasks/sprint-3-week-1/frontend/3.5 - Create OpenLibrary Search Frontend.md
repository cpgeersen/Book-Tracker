# 3.5 OpenLibrary Search Frontend

### Main Objective:
Create the interface that will be used to add books to the local database using OL data.

#### UI:
Will functionally look the same as the functional local search without the filters.

![UI]()

#### Functionality:
Must have similar functionality like local search were it will populate results underneath. The JSON received by
the frontend will be in the same form as the local search JSON schema.

#### Note:
We may need to add some additional features later to accommodate data disparities.
