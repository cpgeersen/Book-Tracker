# 4.3 - Create export/import CSV function


### Main Objectives:
Create the export/import functionality for CSV

### Requirements:
- Must export full database as a CSV
- Must be able to fully reconstruct database from CSV generated

Note: If it becomes too hard to fully export and import with CSV, it would be fine to create functionality to back up
and restore the `bt.db` file

Note: There is not really pseudocode for this task since I am not exactly sure how to go about this. I do not have
a ton of experience with CSV.

### 4.3.1 - Export CSV Function
- Generate a file name by appending the date and time to the file name and put in `app/data/backups`
- Must include all data from the database (except cover images themselves since those are stored in the data folder)

### 4.3.2 - Import CSV Function
- When importing first use the database purge function from Task 4.4 to delete the whole database and rebuild
- Next, using the CSV, process the file to recreate the full database.
