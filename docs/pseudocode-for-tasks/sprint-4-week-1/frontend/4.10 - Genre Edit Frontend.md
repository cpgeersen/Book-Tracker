# 4.10 - Genre Edit Frontend

### Main Objective
Add a genre edit frontend to the individual book page.

### Requirements:
1. First add genre information to the page using flask placeholders.
2. Next, add an 'edit' button next to the genres.
3. When the 'edit' button is clicked, changes the genres to the genre edit interface.
4. Have four selection dropdowns:
   - Genre_1 dropdown: fiction or nonfiction (can be directly put into the page without placeholders)
   - Genre_2 through Genre_4:
     - Make sure the initial selection is the current genre
     - Populate the selection list based on the values from genres.py (Note: For now just add flask placeholders
     with a for-loop to populate this information. Can use mock data if needed.)
     - When the user selects one genre in any of the dropdowns Genre_2 through Genre_4, make sure that value is not
     selectable from the other genre dropdown.
   - Save button near will submit the changes to the server and revert the genres to the initial display
